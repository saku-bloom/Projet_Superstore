- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook

## 📂 Contenu
- `Sample - Superstore.csv` → Données brutes
- `notebook/` → Contiendra l’analyse en Python
- `README.md` → Présentation du projet
